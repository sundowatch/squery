# sQuery (Additional jQuery Functions)

* ```object.hasParent(selector, returnParent=false);```
* ```object.hasChild(selector, returnChild=false);```
* ```object.hasAttr(attrName);```
* ```object.data(attrName, attrValue=false);```
* ```object.eqs(...args);```
* ```object.eqEven();```
* ```object.eqOdd();```
* ```object.near(selector);```
